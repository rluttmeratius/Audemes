Folders:
	audio: the audeme audio files and sound effects used in the game.
	fonts: custom web fonts used in the game.
	img: images used in the game.

Files:
	index.html: access game by opening this file.
	clear-sans.css: css for web fonts.
	style.css: main css for the game.
	jquery-1.11.1.min.js: jquery.
	data.js: stores parameters of the game.
	control.js: game controller.
	sequences.js: stores audeme sequences.
	list.html: use it to view all sequences.
